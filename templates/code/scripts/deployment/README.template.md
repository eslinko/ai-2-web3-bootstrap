# Deployment spec (starter)

This folder holds **human/AI-readable** deployment scenarios, not the node HTTP API.

## Minimum JSON shape

See `pipeline.minimal.example.json`: array of steps with optional `validation` hints.

## Conventions

- Do not commit real private keys.
- Replace scenario names and contract identifiers for your project.
- Reference-only for full multi-scenario matrices — keep those in docs if large.
