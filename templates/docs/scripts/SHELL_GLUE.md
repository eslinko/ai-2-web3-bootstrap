# Shell glue

## Назначение

Короткие bash-скрипты для:

- синхронизации **ABI/артефактов** в consumer-папки (`SOURCE` / `TARGET`);
- **smoke** (curl health node/uploader);
- локальной **оркестрации** без хардкода секретов в теле (только env).

## Что можно переносить как pattern

- `set -e`, переменные по умолчанию через `${VAR:-default}`.
- Параметризуемые **BASE_URL** для node и uploader.
- Отдельный **env file** или экспорт из CI.

## Чего не делать

- Не копировать длинные operator playbooks с **ngrok**, **дефолтными приватными ключами** и **жёсткими путями** эталона как обязательный артефакт.

## Решение

`scripts.shell_glue`: `enabled` | `disabled`.
